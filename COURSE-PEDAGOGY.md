# Patrones Pedagogicos — Sinapsis + Hermes: Domina tus dos agentes AI

## Tesis central del curso

**"No fusionar. Usar cada uno para lo que sirve. Conectar via bridge scripts."**

Esta tesis se refuerza en cada modulo, no solo en la leccion 8 (comparativa). Cada leccion practica demuestra que las herramientas cubren nichos diferentes sin overlap.

## Sistema de niveles alternantes

El curso alterna niveles deliberadamente para mantener el engagement:

| Leccion | Nivel | Justificacion |
|---------|-------|---------------|
| 1. Que son agentes | Basico | Establecer vocabulario comun |
| 2. Instalar Sinapsis | Basico | Reducir friccion de entrada |
| 3. Primera sesion Sinapsis | Basico→Intermedio | Transicion suave |
| 4. Instalar Hermes | Basico | Reducir friccion (2do tool) |
| 5. Comandos Hermes | Basico→Intermedio | Transicion suave |
| 6. Sinapsis a fondo | Avanzado | Profundizar tras base solida |
| 7. Hermes a fondo | Avanzado | Profundizar tras base solida |
| 8. Comparativa | Intermedio | Sintetizar, decidir |
| 9. Scripts bridge | Avanzado | Aplicar integracion tecnica |
| 10. Workflow diario | Intermedio | Contextualizar en la practica |
| 11. Extensiones futuras | Avanzado | Abrir posibilidades |
| 12. Proyecto final | Todos | Integrar todo lo aprendido |

**Patron**: basico-basico-transicion-basico-transicion-avanzado-avanzado-sintesis-avanzado-practica-avanzado-integrador

## Gamificacion

- **5 niveles** que mapean al progreso real del alumno (Novato→Maestro Dual)
- **8 badges** distribuidos: 2 progreso, 2 reto, 2 exploracion, 2 especiales
- **XP diferenciado**: teoria=30, practica=45, reto=25 (+10 bonus primer intento)
- El badge "Arquitecto de Decisiones" refuerza la tesis del curso: el alumno DEBE argumentar por que no fusionar

## Instincts pedagogicos generados

Este curso es el primero del dominio "ai-agents-tooling". Se esperan instincts como:

- "Cuando se comparan 2 herramientas, siempre incluir tabla de escenarios (tarea → herramienta)"
- "Los retos de instalacion necesitan criterios de exito verificables con comandos"
- "Las lecciones avanzadas funcionan mejor con diagramas de flujo de datos que con texto largo"
- "El glosario progresivo evita repeticion y mantiene el contexto limpio"

Estos instincts se promoveran a global si aparecen en 2+ cursos con confidence >= 0.7.
